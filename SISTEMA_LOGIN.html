🔌 Endpoints da API

Método Endpoint Descrição
POST /api/login Autentica um usuário
POST /api/registrar Cadastra cliente (somente analista/admin)
GET /api/perfil Obtém perfil do usuário
GET /api/clientes Lista clientes cadastrados
GET /api/clientes/{id} Obtém cliente específico
GET /api/clientes/estatisticas/resumo Estatísticas gerais
POST /api/analisar-transacao Analisa transação suspeita
